=======================================
Question2Answer Categories
=======================================
-----------
Description
-----------
This is a plugin for **Question2Answer** that allows individual categorizing of uncategorized questions

--------
Features
--------
- categorize questions  individually, bulk submission

------------
Installation
------------
#. Install Question2Answer_
#. Get the source code for this plugin from github_, either using git_, or downloading directly:

   - To download using git, install git and then type 
     ``git clone git://github.com/NoahY/q2a-cat.git cat``
     at the command prompt (on Linux, Windows is a bit different)
   - To download directly, go to the `project page`_ and click **Download**

#. navigate to your site, go to **Admin -> Categories** on your q2a install, then click "Recategorize Individually".
